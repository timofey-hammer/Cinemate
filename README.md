# Cinemate
Приложение для поиска и сохранения фильмов. 

Данное приложение позволяет осуществлять поиск фильмов/сериалов, просматривать их трейлеры, скачивать их. 

## Содержание
- [Технологии](#технологии)
- [Использование](#использование)
- [Источники](#источники)

## Технологии
- Swift 5
- IOS 16.0
- Xcode 14.0
- UIKit only Programmatically
- CoreData
- [SDWebImage](https://github.com/SDWebImage/SDWebImage)
